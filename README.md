# AI-Enabled Flood Detection System

## Overview
This project is a satellite-based flood detection system using deep learning. It processes Sentinel-1 SAR and Sentinel-2 optical imagery to provide flood detection and analysis.

## Features
- Flood detection using satellite imagery
- Support for both Sentinel-1 SAR and Sentinel-2 optical data
- Interactive web interface built with Streamlit
- Deep learning model based on U-Net architecture with ResNet34 encoder
- Detailed visualizations and analytics of flood-affected areas

## Technologies Used
- Python
- PyTorch
- Streamlit
- segmentation-models-pytorch
- rasterio
- NumPy
- Plotly
